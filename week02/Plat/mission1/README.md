# 프로젝트 이름

## 📌 미션

기존 HTML과 JavaScript로 이루어진 TodoList Web Page를 React로 변환하는 작업을 진행했습니다.

---

## 🛠️ 사용 기술

* React
* TypeScript

---

## 📂 구현 내용

* useState를 사용한 list 구현
* HTML코드 컴포넌트화 진행
* useContext를 이용한 props-drilling 해결

---

뭔가 더 생각하면서 하게 되어서 좋은 학습시간이 된 것 같습니다.
